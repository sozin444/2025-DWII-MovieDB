# MyMovieDB - Product Overview

MyMovieDB is a comprehensive movie management system with complete user authentication and security features.

## Core Features

- **User Management**: Complete registration, login/logout, email validation, password reset
- **Security**: 2FA/TOTP authentication, password complexity validation, JWT tokens, encrypted sensitive data
- **User Profiles**: Photo upload with cropping, automatic avatar generation, identicons
- **Movie Database**: Integration with TMDB API for movie data, cast, crew, and metadata
- **Data Management**: Comprehensive seeding system for populating database with movie data

## Target Users

- Movie enthusiasts who want to manage their personal movie collections
- Users requiring secure authentication with 2FA support
- Administrators managing movie database content

## Key Value Propositions

- Secure user authentication with modern 2FA support
- Rich movie metadata from TMDB integration
- Professional-grade security features (encryption, JWT, password policies)
- Comprehensive data seeding and management tools
